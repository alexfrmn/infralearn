# Compress — Сжатие контекста

Когда контекстное окно заполняется, сожми текущий разговор:

## Что сохранить
- Ключевые решения и их причины
- Текущий статус задачи
- Важные находки и ошибки
- Файлы которые менялись

## Что убрать
- Промежуточные попытки и ошибки
- Повторяющийся вывод команд
- Длинные листинги кода (оставить только путь + описание изменений)

## Формат вывода
```
## Сжатый контекст
**Задача:** [одно предложение]
**Статус:** [что сделано / что осталось]
**Ключевые решения:** [список]
**Изменённые файлы:** [пути]
**Следующие шаги:** [1-3 пункта]
```
