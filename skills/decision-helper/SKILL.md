# Decision Helper

Помоги принять решение. Используй структурированный подход:

## Шаги
1. **Переформулируй** проблему одним предложением
2. **Варианты** — перечисли все варианты (2-4)
3. **Критерии** — по каким параметрам сравниваем
4. **Pros/Cons** — таблица для каждого варианта
5. **Рекомендация** — что выбрать и почему
6. **Риски** — что может пойти не так

## Правила
- Не додумывай за пользователя — спрашивай если не хватает данных
- Учитывай контекст из CLAUDE.md
- Отвечай на русском
- Будь кратким но полным
