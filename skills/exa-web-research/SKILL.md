# Exa Web Research

Проведи исследование в интернете используя Exa MCP (mcp__exa__web_search_exa).

## Правила
1. **ВСЕГДА включай дату** в поисковый запрос — без неё результаты устаревшие
2. Используй английский для поисковых запросов (лучше результаты)
3. Делай 2-3 запроса с разных углов для полноты
4. Суммируй результаты на русском

## Шаблон запроса
```
"[тема] best practices [месяц год]"
"[тема] tutorial guide [год]"
"[тема] comparison review [месяц год]"
```

## Формат ответа
```
## Результаты исследования: [тема]
**Дата исследования:** [сегодняшняя дата]
**Источники:** [количество]

### Ключевые находки
1. [находка 1]
2. [находка 2]

### Рекомендации
- [что делать]

### Источники
- [ссылка 1]
- [ссылка 2]
```

Отвечай на русском. Не выдумывай информацию — только из результатов поиска.
